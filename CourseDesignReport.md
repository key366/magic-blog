# 《软件开发课设》项目报告 - 见习魔法师的手记

## 1. 项目概述 (Project Overview)

### 1.1 项目背景
在当前互联网时代，个人博客不仅是技术分享的载体，更是展示个人风格与成长的平台。为了记录 React 技术的学习历程（即“奥术修行”），本项目旨在开发一个具有高度个性化、沉浸式体验的单页应用（SPA）博客系统。

### 1.2 项目目标
- 构建一个功能完备的个人博客系统，支持文章的增删改查（CRUD）。
- 实现数据的本地持久化存储，确保刷新不丢失。
- 打造独特的“魔法/RPG”视觉风格，提供沉浸式用户体验。
- 实现响应式布局，适配 PC 端与移动端访问。

---

## 2. 技术选型 (Technology Stack)

### 2.1 核心框架
- **React 18**: 利用 Hooks（useState, useEffect, useMemo）进行函数式组件开发，实现高效的状态管理与视图渲染。
- **Create React App**: 项目脚手架，快速搭建开发环境。

### 2.2 路由与导航
- **React Router v6**: 实现单页应用的路由跳转，包括嵌套路由（Layout）、动态路由（文章详情页）、路由守卫（登录拦截）。

### 2.3 UI 组件库
- **Ant Design v5**: 使用 Card, List, Form, Modal, Timeline 等高质量组件，并结合 ConfigProvider 实现全局暗黑模式与主题色定制。
- **Ant Design Icons**: 使用丰富的图标库增强视觉表现。

### 2.4 其他工具
- **React Markdown & Remark GFM**: 实现 Markdown 语法的解析与渲染，支持代码高亮与表格。
- **Moment.js**: 处理时间格式化。
- **UUID**: 生成唯一标识符。
- **LocalStorage**: 模拟后端数据库，实现用户数据、文章内容、点赞评论的持久化。

---

## 3. 系统设计 (System Design)

### 3.1 功能模块设计
1.  **文章管理模块**：
    -   **刻录卷轴 (发布)**：支持 Markdown 编辑、封面设置、标签选择。
    -   **重铸卷轴 (编辑)**：回显已有文章内容进行修改。
    -   **销毁卷轴 (删除)**：从列表中移除文章。
    -   **阅读卷轴 (详情)**：渲染 Markdown 内容，展示元数据。

2.  **互动模块**：
    -   **注入魔力 (点赞)**：支持点赞/取消点赞，状态持久化。
    -   **评论功能**：发表评论并实时显示。

3.  **用户模块**：
    -   **法师塔入口 (登录/注册)**：模拟身份验证。
    -   **法师塔 (个人中心)**：展示并编辑个人资料（昵称、职业、简介）。

4.  **个性化模块**：
    -   **魔力色相 (主题换肤)**：支持 6 种元素主题色切换。
    -   **记忆回廊 (归档)**：按月份时间轴展示历史文章。

### 3.2 数据结构设计 (LocalStorage)
-   **`my_blog_posts`**: 存储文章数组（包含 id, title, content, tags, likes, comments 等）。
-   **`my_blog_user`**: 存储当前登录用户信息。
-   **`my_blog_theme_color`**: 存储用户选择的主题色。

## 4. 详细设计与实现 (Implementation)

### 4.1 状态提升 (Lifting State Up)
为了保证数据的一致性，将 `posts` 和 `user` 状态提升至根组件 `App.js` 中管理，并通过 Props 向下传递给 `Home`, `PostDetail`, `WritePost` 等子组件。同时封装 `updatePosts` 函数，确保每次状态更新都同步写入 LocalStorage。

### 4.2 沉浸式魔法主题
-   **视觉风格**：采用深色星空背景 + 金色像素字体 (`VT323`)，配合 CSS 动画实现的魔法粒子与魔法阵背景特效。
-   **文案包装**：将传统术语包装为魔法世界观（如“发布”->“刻录”，“点赞”->“注入魔力”）。

### 4.3 响应式适配
利用 CSS Media Queries 和 Ant Design Grid 系统，针对移动端（<768px）进行了专门适配：
-   导航栏自动折叠为抽屉式菜单。
-   文章列表布局由左右结构调整为上下堆叠。
-   字体与间距自适应调整。
